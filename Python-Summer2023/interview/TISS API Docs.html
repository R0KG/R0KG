Adressbuch 2.2 ¶
get
/api/orgunit/v22/code/:code
Detaildaten einer Orgeinheit auf Basis der Codes
Beschreibung
Detaildaten einer Orgeinheit auf Basis der Codes

Parameter
Name Typ Datentyp Beschreibung
id path string Orgeinheit Codes
recursive query boolean Falls true, werden die Unter-Orgeinheiten rekursiv inkludiert
persons query boolean Falls true, werden die Daten der Mitarbeiter und der Leitung der Orgeinheit im JSON inkludiert
fallback_picture query boolean Base64-Representation des Fallback-Bildes wird geliefert (wenn true). Das Fallback-Bild (derzeit die Initiale oder andere benutzerdefinierten Buchstaben) wird dann angezeigt, wenn der Benutzer kein Profilbild ausgewählt bzw. hochgeladen hat.
preview_picture_uri query boolean Eine skalierte Version des Profilbildes wird geliefert (wenn true). Das gelieferte Feld hat den Wert "null", wenn der Benutzer kein Profilbild ausgewählt bzw. hochgeladen hat.
intern query boolean Auch interne User_innen z.B. rein-Studierende werden angezeigt
Response
HTTP Status Code Beschreibung
200 JSON
404 Orgeinheit mit dem angegebenen Codes existiert nicht
500 Unerwarteter Serverfehler
Beispiel-Aufruf
/api/orgunit/v22/code/E019-03

Beschreibung
Redirect to the researcher's public profile page. Primarily used in the sitemap generated by /sitemap{extension}. The result is configured to be cached for 24 hours. FPF Rest API Documentation

Parameter
Name Typ Datentyp Beschreibung
name path string (required) the researcher's name
id path string (required) the researcher's id
cache query string (optional) default value is 'true', 'false' will disable any cache usage
Response
HTTP Status Code Beschreibung
200 redirect the client to the actual location of the public page in FPL
301 redirect to the public page in FPL
500 configuration error
